🧭 Progression 1 → 41+ (v0.21.7)

The game runs on 5-floor tiers. Every 5th floor acts as a checkpoint floor with teleporter return tile. Deep progression should be done in loops: push, farm, upgrade, re-push.

Tier and Checkpoint Rules

  • Tier formula: tier = floor((max(1,floor)-1)/5)
  • Recommended level: 1 + 4*tier
  • Checkpoint floors: 5,10,15,...
  • Teleporter unlock: first unlock happens at floor 5 and then every +5 reached.

Teleporters and Floor Entry

Teleport choices are generated from unlocked checkpoints only. If max floor is 27, available teleport floors are 5/10/15/20/25.

Checkpoint floors include a town return tile and still maintain danger zones away from stairs/portal to avoid trivial spawn loops.

41+ Endgame Notes

  • Enemy HP/ATK ramps aggressively with tier spike and elite multipliers.
  • Titan ultra variants become meaningful threats; avoid prolonged face-tanking.
  • Epic rarity chance is highest in the game in this range; this is the intended farm loop.
  • If progression stalls: rotate 36-40 farming, then return to 41+ with upgraded loadout.
