Wiki vivant du prototype RPG browser
Jouer Lab

Vision

Créer un action-RPG ultra léger style hack-and-loot, jouable sur mobile et desktop, avec progression persistante et architecture portable vers iOS/Android/Windows/Linux/Steam/Switch.

Boucle de jeu (MVP actuel)

  • Spawn dans le village.
  • Entrée du donjon avec étages progressifs.
  • Combat temps réel simplifié (attaque, dégâts, mort, respawn).
  • XP + level up + loot + or.
  • Sauvegarde locale par utilisateur (Google si connecté, sinon visiteur).

Contrôles

  • Desktop: WASD + click-to-move + Space (attaque).
  • Mobile: D-pad tactile + bouton attaque.
  • Bouton Full screen mobile.

Plan de portabilité

  1. Moteur gameplay isolé (JS pur) sans dépendances lourdes.
  2. Rendu Canvas 2D pixel art léger.
  3. API backend découplées (auth/progression) pour wrapper natif.
  4. Packaging progressif:
    • PWA mobile
    • Tauri/Electron desktop
    • Godot/Unity bridge si besoin console

Roadmap courte

  • Classes de personnages + arbres de talents.
  • Items équipables (slots arme/armure/anneau).
  • Quêtes PNJ et économie village.
  • Instances coop et leaderboard saisonnier.
  • Backend de sauvegarde serveur (cross-device réel).
