✨ Overview

BrowserQuest Online is a persistent browser ARPG focused on short, repeatable sessions: farm, upgrade, push deeper floors, return to town, optimize, repeat.

The game supports mobile + desktop with a shared account progression model. Character state, inventory, and progression are persisted server-side.

Session Loop
Fight → Loot → Upgrade → Push
Drop Model
Low spam, meaningful upgrades
Difficulty Curve
Big step every 5 floors
Late Game
Tier 21+ items + elite mobs

⚙️ Core Systems

Town Portal (3s cast)

Portal cast is interruptible by enemy hits. It acts as a recovery path and anti-stuck fallback. Checkpoint floors include a teleporter tile back to town.

Death penalty

On lethal damage, XP penalty is applied (~10% of level XP requirement). If penalty would drop below zero XP, death is avoided and player survives at low HP.

Lava floors (11+)

Lava tiles deal periodic burn damage and trigger visual warning feedback. Lava generation avoids stairs and teleporter neighborhoods.

Persistence + account safety

Server save is authoritative. Session conflict handling prevents multi-device drift. Recovery flow supports guest-to-Google account migration.

💰 Economy & Merchant

Merchant buy/sell prices are tuned to keep progression meaningful. Potions are cheap consumables; equipment purchases are expensive but strategic.

  • Sell-all-downgrades flow keeps inventory clean.
  • Upgrade indicators compare total item value, not only slot type.
  • Renamer NPC allows nickname change (gold fee + content filter).

🌐 MMO, Chat, Safety

  • Global presence + chat with anti-spam/bad-word filtering.
  • Chat supports persistent history and ticker preview in game HUD.
  • UI is tuned for mobile portrait + landscape and desktop.

■ Abusive messages and offensive names are blocked by server-side filters.

🧠 Strategy by Stage

Floors 1-10

Prioritize consistent upgrades and survivability. Learn movement and disengage timing.

Floors 11-20

Respect lava and ranged pressure. Farm efficiently when blocked. Avoid risky multi-pulls.

Floors 21+

Need stronger tier items and cleaner build management. Titans/archons are gear checks, not mandatory fights.
